Detailed Description

This structure contains the state of a single instance of a SoftUART module.

Field Documentation

void(* pfnIntCallback)(void)

The address of the callback function that is called to simulate the interrupts that would be produced by a hardware UART implementation. This address can be set via a direct structure access or using the SoftUARTCallbackSet function.
