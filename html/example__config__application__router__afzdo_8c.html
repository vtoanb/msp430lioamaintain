Detailed Description

Resets Module, configures this device to be a Zigbee Router, joins a network, then sends a message to the coordinator periodically. Also periodically requests new configuration data from the Coordinator.

Demonstrates how to implement two-way control of a remote node, and also how to do both send and receive of messages in the same state machine.

If you need assistance we're here to help; see below for support options. Tesla has extensive experience with sensor-to-server applications and everything inbetween.

Note:
This matches example_config_application_coordinator.c or can just be used with the Gateway
