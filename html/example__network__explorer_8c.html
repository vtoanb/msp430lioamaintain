Detailed Description

Allows the user to configure this device as either a Coordinator, Router, or End device and explore various features of the Zigbee Module, such as sending messages via short address or long address, finding devices, etc. Uses the AF/ZDO interface. Uses a command line interface (CLI).

Note:
Different functions can be turned on/off by defining the compile time options. While the functions will work on various processors, due to flash space restrictions you may not be able to use all functions on a smaller processor (like the MSP430 value line). The best tool to evaluate this Network Explorer is the Stellaris LaunchPad with Code Composer Studio. The Stellaris LaunchPad is able to run all features of the Network Explorer and is hard to beat for < $5US!

void(* debugConsoleIsr)(int8_t) 
This is a function pointer for the Interrupt Service Routine called when a debug console

character is received. To use it, declare it with extern void (*debugConsoleIsr)(int8_t); and then point it to a function you created, e.g. debugConsoleIsr = and your function handleDebugConsoleInterrupt() will be called when a byte is received on the debug serial port. Obviously your handleDebugConsoleInterrupt must have the same method signature as the function pointer. In other words, your function should be declared as: void handleDebugConsoleInterrupt(int8_t);
