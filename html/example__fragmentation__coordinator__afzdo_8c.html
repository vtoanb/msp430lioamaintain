Detailed Description

Resets Module, configures this device to be a Zigbee Coordinator, and displays any messages that are received. If the message is extended then gets all the parts and displays it. Uses the AF/ZDO interface.

This matches example_fragmentation_ROUTER.c Get this running before the router, or else the router won't have anything to join to.
