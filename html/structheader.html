Field Documentation

uint8_t flags 
Flags to indicate various stuff; dependent on application.

One might be for a pending firmware update, another might be to indicate a low battery.

uint8_t mac[8] 
MAC Address of sender (for messages from device to server) or of destination (for messages

from server to device). Mainly used for messages going through a gateway (e.g. Zigbee to Ethernet) so that the server receiving the message knows who sent the message. While this can be obtained by querying the network, it's faster to put it in the message

uint16_t sequence

Application level sequence number, used to track response from the server etc. LSB first

uint8_t version 
Protocol version. If message protocol changes then use a different number for version and

if/then on receiver accordingly
