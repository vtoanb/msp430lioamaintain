Detailed Description

Resets Module, configures this device to be a Zigbee Coordinator, and displays any messages that are received. If the message contains a known object-identifier (OID) then parses the contents of the OID. Also replies to incoming "Config Request" messages with a "Config Response" message containing the current color settings. Pushing the button will change which color shall be displayed by all nodes.

Uses the AF/ZDO interface.
