file  example_per_tester_coordinator.c
 Packet Error Rate Tester: configures this device to be a Zigbee Coordinator and counts the number of messages received. Stops once 1000 messages are received. LED 0 will toggle each time a message is received. Pressing the button will display the message count. Uses the AF/ZDO interface.

 Packet Error Rate Tester: Resets Radio, configures this device to be a Zigbee Router, joins a network, then sends 1000 messages to coordinator. Pressing the button will restart the application.
