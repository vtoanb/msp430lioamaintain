void displayMessage ( )

Displays the type of message in zmBuf. Ignores the message if length = 0.

Precondition:
moduleHasMessageWaiting() is true
getMessage() called to get message into zmBuf
Postcondition:
zmBuf Length field = 0
moduleResult_t expressStartModule ( const struct moduleConfiguration *  mc,
const struct applicationConfiguration *  ac,
const uint8_t  moduleRegion 
)

Starts module using an operating region as a parameter. This does NOT read the GPIO pin to set the region.

Parameters:
mcthe module configuration - what RF channel, which PAN ID, etc. These options are used in this expressStartModule function as arguments to the various functions.
acthe Zigbee application configuration - which endpoint to use and other global settings. If not using GENERIC_APPLICATION_CONFIGURATION then you must #define the compilation option SUPPORT_CUSTOM_APPLICATION_CONFIGURATION. Normally this option is NOT defined to reduce code size.
moduleRegion- which region of the world to use to ensure FCC/ETSI compliance.
moduleResult_t startModule ( const struct moduleConfiguration *  mc,
const struct applicationConfiguration *  ac 
)

Start the Module and join a network, using the AF/ZDO interface. Reads RF Operating Region (US/EU) from GPIO.

Parameters:
moduleConfigurationthe settings to use to start the module
applicationConfigurationthe settings to use to start the Zigbee Application

module.c for more information about each of these steps.
